About Questions Import
----------------------

Questions Import is an addon module for quiz. It provides the feature to create a bulk of questions from a text file. The import text file suppose to have a N number questions in some predefined format.

Some of the predefined formats are as follow
1. aiken
2. gift
3. learnwise
4. webct
5. csv    
5. hotpot
6. qti

The sample import files were avilable in "./examples" directory. Feel free to create an issue here http://drupal.org/node/add/project-issue/quiz for support request or bug reporting.
